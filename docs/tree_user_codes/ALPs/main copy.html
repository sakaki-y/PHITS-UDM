Descriptions

以下のラグランジアンで記述されるALPsのPrimakoff過程に伴う生成と光子対への崩壊を扱う。

\(\delta\mathcal{L}= -\frac{1}{4} g_{a\gamma\gamma} a F_{\mu\nu} F^{\mu\nu} +\frac{1}{2} (\partial_\mu a)^2 -\frac{1}{2} m_a^2 a^2\)

ただし、 \(a\) は ALPs、 \(m_a\) は ALPs mass、 \(g_{a\gamma\gamma}\) は ALPs-photon 結合定数、 \(F_{\mu\nu}\) is the strength of the photon field.
