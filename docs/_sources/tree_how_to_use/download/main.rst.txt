########################################
ダウンロード
########################################

| :download:`phits-udm-20230120.zip` (最新)
| :download:`phits-udm-20230109.zip`

ダウンロード後、[:ref:`sec_1`] を参照して PHITS-UDM の実行ファイルを作成して下さい。

|

**（ソフトウェア要件）**

* PHITS 3.29 以上

* Fortran コンパイラ（Intel Fortran や gfortran など）

* Python3 または Python2

    * Python は、オリジナルのソースファイルのコピーと変換を行うためだけに使用される。変換された 10 ファイルを直接配布することで Python の使用は原理的にはスキップできる。
